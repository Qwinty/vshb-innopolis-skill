# vshb-innopolis

Небольшой skill-пакет с документами по Весенней школе бакалавров Innopolis 2026.

Что внутри:
- логистика и трансфер;
- расписание по дням;
- чек-лист участника;
- формы для заселения;
- оригиналы документов и обработанные Markdown-версии.

Структура:
- `SKILL.md` — краткие инструкции и навигация;
- `references/` — обработанные документы для быстрого чтения;
- `assets/originals/` — исходные PDF и DOCX.

Подходит для задач:
- быстро найти время/место события;
- понять, как добраться до Иннополиса;
- проверить, что взять с собой;
- открыть формы для заселения.
