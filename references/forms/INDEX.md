# Формы для заселения

Оригиналы форм лежат в `assets/originals/forms/`.

Файлы:
- `Регистрационная карта 2026.docx`
- `Согласие на обработку персональных данных (совершеннолетние).docx`
- `Согласие на распространение.docx`

Используй эти файлы, когда нужно:
- перечислить, какие формы требуются;
- открыть/конвертировать форму;
- помочь с заполнением;
- напомнить, какие поля нужно заполнить.

Важно:
- не придумывай поля, которых не видел в документе;
- если нужно точное содержимое формы, сначала открой сам файл.
