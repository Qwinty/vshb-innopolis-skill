## Какие документы нужно привезти с собой?

### Для заселения необходимы

- заполненная регистрационная карта;
- согласие на обработку персональных данных (кампус);

## Кампус

Кампус является общежитием для сотрудников и студентов Университета, поэтому бытовые приборы и посуда не предоставляются

Проживание в кампусе включает в себя несколько типов расселения:

- двухместный однокомнатный номер
- четырехместный двухкомнатный номер
- пятиместный трехкомнатный номер

В каждом номере находится холодильник и кухня (без посуды). Банные полотенца для гостей предусмотрены.

На каждом этаже в кампусе есть прачечная, которая включает в себя стиральную машину, сушильную машину, утюг, гладильную доску.

## Что необходимо взять с собой?

- повседневную одежду;
- гигиенический набор (шампунь, гель для душа, зубная паста, зубная щетка);
- личную медицинскую аптечку (если есть необходимость);
- фен;
- спортивную форму и обувь для тренажерного зала;
- плавательные принадлежности и полотенце
